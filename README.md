# Release Notes

## **v1.0 – Initial Release**
- Basic calculator tool.
- Supports simple addition of two numbers.
- Includes a minimal command-line interface.
- This release serves as the baseline for future improvements.

---

## **v1.1 – Feature Update**
- Added **factorial** calculation feature.
- Added **input validation** to handle incorrect user inputs gracefully.
- Improved program stability and user experience.
- Updated documentation and cleaned up code formatting.
